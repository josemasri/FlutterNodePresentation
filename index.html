Servelet

  • Respuesta de Java a PHP
  • Lanzado en 1996 y en 1997 parte de JDK
  • Desarrollo de aplicaciónes web utilizando Java

Funcionamiento de Servelet

  1. El cliente envía una petición al servidor web.
  2. El servidor recibe la petición.
  3. El servidor delega la petición a su correspondiente Servlet.
  4. El Servlet realiza un procesamiento y genera una respuesta.
  5. El servlet envía esta respuesta al servidor web.
  6. El servidor web envía la respuesta de nuevo al cliente y navegador del cliente le despliega la respuesta en su pantalla al usuario.

Estructura de carpetas

  1. ".dart_tool/": Librerías de pub instaladas
  2. ".idea/": Archivos y configuraciones para el IDE de Android Studio.
  3. "android/": Proyecto completo de Android nativo.
  4. "ios/": Proyecto completo de IOS nativo.
  5. "lib/": Nuestro código fuente dart
  6. "lib/": Realizar pruebas automátizadas
  7. "pubspec.yaml": Paquetes de terceros que se desean utilizar, configuraciónes, fuentes, imágenes estáticas.
